# zephyr

Zephyr comes from Greek, which means "breeze", implying that this is a lightweight tool.

Zephyr 来自希腊语，表示“微风”，暗示着这是一个轻量级的工具。

## aria2go

**Go bindings for aria2**

- aria2.addUri
- aria2.tellStatus
- aria2.tellStopped
- aria2.getGlobalStat
- system.listMethods

## zexec

**os/exec wrapper**

- StartProcess & StopProcess
- ffmpeg command export
